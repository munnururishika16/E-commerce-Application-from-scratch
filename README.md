
E-Commerce Website
Project Overview

This is an E-Commerce web application developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP.

Features
User Registration and Login
Product Listing
Add to Cart
Order Management
Responsive Design
Database Integration
Technologies Used
PHP
MySQL
HTML5
CSS3
JavaScript
XAMPP
Installation Steps
Install XAMPP.
Copy the project folder into htdocs.
Start Apache and MySQL from XAMPP.
Import the database file into phpMyAdmin.
Open http://localhost/ecommerce in your browser.
Database

Import the provided .sql file into phpMyAdmin before running the project.
